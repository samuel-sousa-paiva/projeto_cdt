💈 SMD Barbearia

Site moderno de barbearia desenvolvido para oferecer uma experiência elegante e prática para os clientes. O projeto conta com páginas de agendamento, login, serviços e um design responsivo com foco em estilo e usabilidade.

🔥 Sobre o Projeto

A SMD Barbearia foi criada para representar uma barbearia moderna, com visual profissional e sistema de navegação simples. O site possui foco em agendamentos online e apresentação dos serviços da barbearia. Inspirado em tendências atuais de design para barbearias premium.
